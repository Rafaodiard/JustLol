# 🎮 JustLol - League of Legends Stats Analyzer

App para analizar estadísticas de League of Legends y publicarla al mercado.

## 📋 Project Overview

Plataforma integral para consultar y analizar estadísticas de jugadores de LoL en tiempo real, con insights avanzados y comparativas.

## 🚀 Quick Start

```bash
git clone https://github.com/Rafaodiard/JustLol.git
cd JustLol
npm install
npm run dev
```

## 📊 Tech Stack

- **Frontend:** React + TypeScript + TailwindCSS + Vite
- **Backend:** Node.js + Express + PostgreSQL
- **Cache:** Redis
- **External API:** Riot Games API
